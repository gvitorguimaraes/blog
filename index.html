olá, seja bem vindo!

Me chamo Gabriel Vitor, sou desenvolvedor de software formado em Sistemas de Informação pela PUC Minas, com mais de dois anos de experiência no mercado. Meus principais interesses são arquitetura de software, sistemas embarcados, projetos DIY (Do It Yourself), ciências da natureza, filosofia e ficção científica. Neste site, você encontrará meus projetos, posts com discussões e pesquisas sobre meus temas de interesse, e minhas informações de contato.

ferramentas e tecnologias

Atualmente meu foco profissional é na área de desenvolvimento back-end, utilizando principalmente a linguagem de programação Java e seu ecossistema de frameworks, como Spring, Spring Boot e a plataforma Jakarta EE. Também trabalho com tecnologias complementares, como o framework Angular para o desenvolvimento front-end, bancos de dados relacionais, como PostgreSQL, entre outras ferramentas.
